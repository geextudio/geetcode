# 极客诚品的代码刻意练习

![logo](./images/geextudio.png)

## 项目目的

* 学习和练习算法、设计模式知识，以及相关的延展内容和理论整理.
* 根据不同语言环境，提供完整的代码开发与测试架构，便于快速开始各类代码或算法实践，而不需要额外花很多时间与搭建环境相搏斗:

  * Python 3.7+
  * Java
  * C# in .NET Core 3.x
  * Javascript
